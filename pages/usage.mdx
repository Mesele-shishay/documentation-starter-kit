# Getting Started

## Installation

Install the SDK using Composer:

```bash
composer require mesasdk/php-mpesa
```

## Basic Usage

Here's a quick example of how to use the SDK for an STK Push request:

```php
use MesaSDK\PhpMpesa\Mpesa;

// Initialize the SDK
$mpesa = new Mpesa([
    'consumer_key' => 'your_consumer_key',
    'consumer_secret' => 'your_consumer_secret',
    'environment' => 'sandbox', // or 'production'
    'shortcode' => 'your_shortcode',
    'key' => 'your_passkey'
]);

// Initiate STK Push
try {
    $response = $mpesa->querySTKStatus('checkoutRequestId');

    // Handle the response
    print_r($response);
} catch (MpesaException $e) {
    // Handle any errors
    echo $e->getMessage();
}
```
