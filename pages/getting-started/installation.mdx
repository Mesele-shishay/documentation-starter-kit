# Installation Guide

This guide will walk you through the process of setting up the project in your local environment.

## Prerequisites

Before you begin, ensure you have the following installed:

- PHP 8.0 or higher
- Composer
- Git

## Installation Steps

1. Clone the repository:

```bash
git clone https://github.com/yourusername/yourrepository.git
cd yourrepository
```

2. Install dependencies:

```bash
composer install
```

3. Configure your environment:

```bash
cp .env.example .env
```

4. Update the `.env` file with your configuration settings.

## Verification

To verify your installation:

1. Run the test suite:

```bash
./vendor/bin/phpunit
```

2. Start the development server:

```bash
php -S localhost:8000 -t public
```

## Common Issues

### Composer Dependencies

If you encounter issues with Composer dependencies:

```bash
composer clear-cache
composer update
```

### Permission Issues

If you encounter permission issues:

```bash
chmod -R 755 storage
chmod -R 755 bootstrap/cache
```
